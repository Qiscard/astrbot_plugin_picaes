# 图片加解密插件 v3.0

图片马赛克加密/解密，支持图片/PDF/ZIP 多种发送方式，可选密码保护。

## 功能特性

- 🔒 **加解密引擎**：本地处理 + API 双模式，10 级加密强度
- 📷 **图片直发**：直接发送或压缩后发送（可配置）
- 📄 **解密转 PDF**：解密后转加密 PDF 文件发送
- 📦 **解密转 ZIP**：解密后转加密 ZIP 文件发送
- 🔗 **图床链接**：API 处理后返回链接（需配合 URL 模式）
- 🔑 **密码控制**：可选是否加密、是否发送密码
- 🗜️ **压缩开关**：可关闭压缩，解密返回原图

## 安装

1. 将插件放入 AstrBot 的 `data/plugins` 目录
2. 后台重载插件，依赖自动安装（`aiohttp` `Pillow` `pyzipper` `pymupdf`）

## 配置

AstrBot 后台 → 插件管理 → 插件配置：

| 配置项 | 默认值 | 说明 |
|-------|--------|------|
| 结果发送方式 | 直接发送图片 | 图片/PDF/ZIP |
| 是否加密 | ✅ | PDF/ZIP 是否设密码 |
| 文件加密密码 | `123` | 加密密码 |
| 是否发送密码 | ✅ | 关闭则提示「文件已加密」 |
| 是否压缩 | ✅ | 关闭则解密返回原图 |
| 图片处理方式 | 本地优先+API回退 | 本地/仅本地/仅API/URL模式 |
| API 接口地址 | `https://picace.995456.xyz/api/process` | 仅 API 模式 |
| 默认密钥 | `picaes` | 未指定时使用 |
| 默认等级 | `4` | 1-10 |
| API 超时 | `120` 秒 | 仅 API 模式 |

## 指令

| 指令 | 说明 |
|------|------|
| `加密 [等级] [密钥]` | 加密图片 |
| `解密 [等级] [密钥]` | 解密（按配置发送） |
| `解密pdf [等级] [密钥]` | 解密后转 PDF |
| `解密zip [等级] [密钥]` | 解密后转 ZIP |
| `图解帮助` | 显示帮助 |

**用法**：发送指令附带图片，或回复图片后发送指令。

## 加密等级

| 等级 | 网格 | 说明 |
|-----|------|------|
| 1 | 10×10 | 最低 |
| 4 | 40×40 | 默认 |
| 7 | 70×70 | 较高 |
| 10 | 100×100 | 最高 |

## 注意事项

- 加密/解密必须使用**相同等级和密钥**
- 发送图片建议选择「文件」形式，避免平台压缩
- 关闭压缩可获得原图质量，但文件更大
- PDF/ZIP 以文件形式发送，可绕过 QQ 图片内容审核
